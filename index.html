Luminance

A Philips Hue client for Linux written in Python and GTK+.

View on GitHub Download .zip Download .tar.gz

Luminance is a client for the Philips Hue lighting system. Its aim is to be beautiful and easy to use yet offer all of the functionality that the Hue system has to offer.

Installation

If you're just looking to use Luminance, you'll want to install from distribution packages. Currently, Luminance is available from the following distributions:

  • None yet, AUR package will be available shortly

Problems

If you are having a problem, please report an issue detailing your environment as much as possible (Distribution, Python version, GTK version, traceback if any, etc).
